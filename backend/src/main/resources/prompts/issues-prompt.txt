以下是某学生在一项实训任务中的全部评价结果（含AI评分和评语）：

{evaluationResults}

请从评语中提炼各类问题，按以下JSON格式输出。仅输出JSON，不输出任何其他内容：
{
  "functional": [{"issue": "问题描述", "severity": "严重/一般/建议", "source": "指标名"}],
  "technical": [{"issue": "问题描述", "severity": "严重/一般/建议", "source": "指标名"}],
  "compliance": [{"issue": "问题描述", "severity": "严重/一般/建议", "source": "指标名"}],
  "innovation": [{"issue": "问题描述", "severity": "严重/一般/建议", "source": "指标名"}]
}

规则：
- functional: 与功能完成度相关的问题（功能缺失、逻辑错误、场景不匹配等）
- technical: 与技术实现相关的问题（架构不合理、代码不规范、性能不足等）
- compliance: 与规范性、文档相关的问题（材料缺失、格式混乱、命名不规范等）
- innovation: 与创新性相关的问题（实现同质化、无亮点、完全照搬基础方案等）
如某类没有可提炼的问题，返回空数组。
